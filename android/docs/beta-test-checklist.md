# βテスト 項目表（Privacy Camera）

対象ブランチ: `claude/android-camera-pii-masking-io6ucl`
作成日: 2026-06-24

## 使い方
- `結果` 欄に `Pass` / `Fail` / `-`（未実施）を記入してください。
- `Fail` の場合は `備考` に再現手順・端末・スクショの所在を残してください。
- フレーバー列: **共通** = Lite/Pro 両方で実施 / **Lite** / **Pro**。
- Lite と Pro は別 applicationId（`com.privacycamera.lite` / `com.privacycamera`）なので
  **同一端末に両方インストールして**並行テストできます。

## 既知の制約（β参加者へ周知）
- **Lite の移行書き出しは平文**：マスク前のオリジナルがそのまま含まれます。取り扱い注意。
- **Lite→Pro の移行は累計30枚まで**：削除しても枠は戻りません。
- **復元はカスタムマスクを持ち越しません**：バックアップは原本を運ぶため、復元後の表示は
  安全側の全体モザイクになります（必要なら再度マスク編集）。
- **パスフレーズを忘れると復元不可**：暗号化バックアップは復号できません。
