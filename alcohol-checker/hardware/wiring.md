# 配線図

## ESP32-DevKitC ピン配置

```
                    ESP32-DevKitC
                  ┌─────────────┐
             3V3 ─┤ 3V3     GND ├─ GND (共通グランド)
             GND ─┤ GND     D23 ├─
                  │         D22 ├─ OLED SCL
                  │  ～省略～  │
              EN  │             │
                  │         D21 ├─ OLED SDA
             D34 ─┤ D34     D19 ├─
     (MQ3 AOUT) │         D18 ├─
                  │         D5  ├─
             D27 ─┤ D27     TX  ├─
      (ブザー) │         RX  ├─
             D26 ─┤ D26     D25 ├─ 緑 LED
       (赤 LED) │         D24 ├─
                  │         D4  ├─
                  │         D2  ├─
                  │         D15 ├─
                  └─────────────┘
```

## 各モジュールの接続

### MQ-3 アルコールセンサー
| MQ-3 ピン | 接続先 |
|-----------|--------|
| VCC | ESP32 3V3 |
| GND | ESP32 GND |
| AOUT (アナログ出力) | ESP32 GPIO 34 |
| DOUT (デジタル出力) | 未接続 |

### OLED ディスプレイ (SSD1306 I2C)
| OLED ピン | 接続先 |
|-----------|--------|
| VCC | ESP32 3V3 |
| GND | ESP32 GND |
| SDA | ESP32 GPIO 21 |
| SCL | ESP32 GPIO 22 |

### 緑 LED (合格表示)
```
ESP32 GPIO 25 → [220Ω抵抗] → LED アノード (長い足)
                              LED カソード (短い足) → GND
```

### 赤 LED (不合格表示)
```
ESP32 GPIO 26 → [220Ω抵抗] → LED アノード (長い足)
                              LED カソード (短い足) → GND
```

### 圧電ブザー (受動型)
```
ESP32 GPIO 27 → ブザー + 端子
                ブザー - 端子 → GND
```

### バッテリー回路
```
リポバッテリー (+) → TP4056 BAT+
リポバッテリー (-) → TP4056 BAT-
TP4056 OUT+        → ESP32 VIN (or 5V)
TP4056 OUT-        → GND
```

## 注意事項

1. **3V3 を使用**: ESP32 の ADC は 3.3V 系のため、MQ-3 は 3.3V で駆動します。  
   （MQ-3 は本来 5V ですが、3.3V でも動作します。感度が若干低下するため R0 校正が重要です）
2. **アナログ GPIO**: GPIO 34 は入力専用です。出力には使用しないでください。
3. **ブザー**: 受動型ブザーを使用してください（`tone()` 関数で音程を制御するため）。
